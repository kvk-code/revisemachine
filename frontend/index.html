🐦 GetIt

Save tweets to your GitHub repository as markdown files

📖 First-Time Setup (Do This Once)

1
Create your own copy of GetIt
Go to GetIt repo → Click "Use this template" → Name it (e.g., my-saved-tweets) → Set to Private
2
Get your Twitter API Key
Sign up at twitterapi.io and copy your API key (~$0.15 per 1000 tweets)
The frontend will automatically save it as a GitHub Secret for you!
3
Create a Fine-Grained PAT (scoped to your repo only)
Create token → Repository access: Only select repositories → Select your repo → Permissions: Contents: Read & Write, Secrets: Read & Write, Actions: Read & Write

✅ Done? Now fill in the form below to start saving tweets!

⚙️ Your Configuration

Credentials are stored only in your browser (localStorage) and never sent to any server.

Example: johndoe/saved-tweets

📥 Save a Tweet

Supports twitter.com and x.com URLs

🔍 How It Works

1

Your credentials are stored only in your browser's localStorage - never sent to any server.

2

When you save a tweet, this page triggers a GitHub Action in your repository.

3

The GitHub Action fetches the tweet using twitterapi.io and creates a markdown file.

4

The markdown file is committed to your repository's tweets/ folder.

🔒 Privacy First: This page runs entirely in your browser. No data is sent to any third-party server.

Your tokens are stored in localStorage and used only to communicate with GitHub's API.
